# DFS на Oracle 6.6

Install keyutils:

```
yum install keyutils
```

Append the following in /etc/request-key.conf:

```
create cifs.spnego * * /usr/sbin/cifs.upcall -c %k
create dns_resolver * * /usr/sbin/cifs.upcall %k
```

Install cifs-utils:

```
yum install cifs-utils
```
