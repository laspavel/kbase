# resize2fs 1.43.4 #

Используется для resize ext4 раздела больше 16Тб

### Сборка ### 

    git clone https://git.atbmarket.com/util/e2fsprogs.git
    cd e2fsprogs
    ./configure
    make

### На выходе ###

e2fsck/e2fsck - проверка ФС
resize/resize2fs - resize ФС
misc - 1,5 дюжины разных утилит разного назначения
